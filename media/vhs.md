Create a terminal GIFs demo as code using [vhs](https://github.com/charmbracelet/vhs)

```bash
# macOS or Linux
brew install vhs

# Arch Linux (btw)
pacman -S vhs

# Nix
nix-env -iA nixpkgs.vhs

# Windows using scoop
scoop install vhs
```

Then, create a `demo.tape` file and run:

```bash
vhs demo.tape
```
